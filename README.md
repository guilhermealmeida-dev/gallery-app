## 📖 Sobre

O **Gallery-App** é uma aplicação de gerenciamento de imagens em nuvem desenvolvida para oferecer uma experiência moderna, rápida e segura no armazenamento e organização de fotografias.

A plataforma permite que usuários autenticados realizem upload de imagens, organizem seus arquivos e os acessem de qualquer dispositivo conectado à internet.

---

## ✨ Funcionalidades

- 📤 Upload de imagens
- 🖼️ Visualização em galeria
- 📂 Organização por álbuns
- 🔍 Pesquisa de imagens
- ❤️ Favoritos
- 🗑️ Exclusão de imagens
- 🔐 Autenticação JWT
- ☁️ Armazenamento em nuvem
- 📱 Interface responsiva
